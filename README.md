# Escritor-I.A.T-count
Un editor de texto diseñado para escritores que buscan inspiración y comodidad. Incluye varios modos creativos que ayudan a desbloquear ideas, un reloj integrado para controlar sesiones de escritura y un modo oscuro completamente negro que cuida la vista y realza la concentración. Su diseño es elegante, minimalista y muy agradable de usar, 

El editor ofrece una experiencia visual pensada para estimular la creatividad del escritor. Cuenta con tres modos principales, cada uno con una atmósfera distinta:

# Modo Lluvioso
Presenta un efecto blanco suave que imita pequeñas gotas cayendo. Es ideal para quienes buscan un ambiente sereno, casi meditativo, que acompaña escenas emotivas o reflexivas.

# Modo Luciérnagas
En este modo aparecen puntos amarillos que titilan como si fueran pequeñas luciérnagas flotando alrededor del texto. Aporta un aire mágico y cálido, perfecto para relatos fantásticos o momentos íntimos.

# Modo Galáctico
Genera partículas blancas que se desplazan sutilmente, dando la sensación de estar entre estrellas. Es un entorno inspirador para ideas profundas, ciencia ficción o capítulos que necesitan un toque de inmensidad.
Para elegir cualquiera de estos modos, el editor usa tres botones de colores ubicados de forma accesible:

Azul para el modo lluvioso
Amarillo para el modo luciérnaga
Rojo para el modo galáctico
# Todo es intuitivo: basta con pulsar un color y el ambiente del editor cambia al instante.
En la parte inferior se encuentra un botón tipo check, pensado para intensificar la experiencia. Al activarlo:
Aparece un reloj en rojo, útil para controlar tiempos de escritura o escenas con límite.
El color del efecto del modo actual también se vuelve rojo, unificando el ambiente.
Tanto el reloj como el efecto entran en un parpadeo sincronizado, creando una sensación de tensión ideal para escribir momentos críticos, combates, suspenso o capítulos de alto impacto emocional.
Finalmente, el editor incluye un botón circular gris que permite alternar entre modo claro y modo oscuro. Con un solo toque, puedes elegir un fondo luminoso para trabajar de día o un fondo negro profundo para proteger la vista durante sesiones nocturnas.
